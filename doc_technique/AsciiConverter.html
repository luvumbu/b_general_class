Classe AsciiConverter

La classe AsciiConverter permet de convertir des chaînes de caractères en leur équivalent ASCII et inversement. Elle utilise deux méthodes statiques :

  • asciiToString pour convertir une chaîne de valeurs ASCII en chaîne de caractères.
  • stringToAscii pour convertir une chaîne de caractères en valeurs ASCII.

Méthodes

1. asciiToString($asciiString)

Cette méthode convertit une chaîne de valeurs ASCII en une chaîne de caractères.

Paramètre :

  • $asciiString (string) : Une chaîne de valeurs ASCII séparées par des virgules. Par exemple, "72,101,108,108,111".

Retourne :

  • (string) : La chaîne de caractères correspondant aux valeurs ASCII. Par exemple, "Hello".

Détails :

La méthode divise la chaîne $asciiString en un tableau de valeurs ASCII en supprimant les espaces inutiles. Elle vérifie que chaque valeur est bien un nombre avant de la convertir en caractère via la fonction chr().

Exemple d'utilisation :

$asciiString = "72, 101, 108, 108, 111";
echo AsciiConverter::asciiToString($asciiString); // Affiche "Hello"

2. stringToAscii($string)

Cette méthode convertit une chaîne de caractères en valeurs ASCII.

Paramètre :

  • $string (string) : Une chaîne de caractères à convertir en valeurs ASCII. Par exemple, "Hello".

Retourne :

  • (string) : Une chaîne de valeurs ASCII séparées par des virgules. Par exemple, "72,101,108,108,111".

Détails :

La méthode parcourt chaque caractère de la chaîne $string et utilise la fonction ord() pour obtenir la valeur ASCII correspondante. Les valeurs ASCII obtenues sont ensuite jointes avec des virgules pour former une chaîne.

Exemple d'utilisation :

$string = "Hello";
echo AsciiConverter::stringToAscii($string); // Affiche "72,101,108,108,111"

Exemple d'utilisation complet :

$asciiString = "72, 101, 108, 108, 111";
$string = "Hello";

// Conversion de ASCII à chaîne de caractères
echo AsciiConverter::asciiToString($asciiString); // Affiche "Hello"

// Conversion de chaîne de caractères à ASCII
echo AsciiConverter::stringToAscii($string); // Affiche "72,101,108,108,111"

Fonctions utilisées

  • chr(int $ascii) : Convertit une valeur ASCII en caractère.
  • ord(string $char) : Convertit un caractère en valeur ASCII.
  • explode(string $delimiter, string $string) : Divise une chaîne de caractères en un tableau.
  • implode(string $glue, array $pieces) : Joint les éléments d'un tableau en une chaîne de caractères, avec un séparateur.
  • trim(string $string) : Supprime les espaces au début et à la fin.
