Documentation Technique pour `ImageResizer`

Classe `ImageResizer`

La classe ImageResizer est utilisée pour redimensionner des images tout en maintenant leur rapport d'aspect. Elle prend en charge les formats JPEG, GIF et PNG.

Méthodes

Constructeur

public function __construct($filename)

Description: Charge l'image à partir du fichier spécifié et récupère ses dimensions.

Paramètres:

  • $filename (string): Le chemin du fichier image à charger.

load($filename)

private function load($filename)

Description: Charge l'image et récupère ses dimensions et son type.

Paramètres:

  • $filename (string): Le chemin du fichier image à charger.

resize($maxWidth, $maxHeight)

public function resize($maxWidth, $maxHeight)

Description: Redimensionne l'image tout en conservant le rapport d'aspect.

Paramètres:

  • $maxWidth (int): La largeur maximale souhaitée.
  • $maxHeight (int): La hauteur maximale souhaitée.

save($filename, $imageType = IMAGETYPE_JPEG, $compression = 75, $permissions = null)

public function save($filename, $imageType = IMAGETYPE_JPEG, $compression = 75, $permissions = null)

Description: Sauvegarde l'image redimensionnée dans un fichier.

Paramètres:

  • $filename (string): Le chemin du fichier où l'image sera sauvegardée.
  • $imageType (int): Le type d'image (par défaut JPEG).
  • $compression (int): Le niveau de compression (pour JPEG).
  • $permissions (int|null): Les permissions du fichier (optionnel).

output($imageType = IMAGETYPE_JPEG)

public function output($imageType = IMAGETYPE_JPEG)

Description: Affiche l'image directement dans le navigateur.

Paramètres:

  • $imageType (int): Le type d'image (par défaut JPEG).

__destruct()

public function __destruct()

Description: Libère la mémoire utilisée par l'image.

Exemples d'Utilisation

<?php
// Création d'une instance de ImageResizer
$resizer = new ImageResizer('path/to/image.jpg');

// Redimensionnement de l'image
$resizer->resize(800, 600);

// Sauvegarde de l'image redimensionnée
$resizer->save('path/to/resized_image.jpg');

// Affichage de l'image directement dans le navigateur
$resizer->output();
?>
