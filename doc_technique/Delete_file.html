Documentation Technique de la Fonction delete_file

Description Générale

La fonction delete_file permet de supprimer un fichier spécifié par son chemin. Elle vérifie si le fichier existe avant de tenter de le supprimer.

Fonction

delete_file($file_path)

Supprime le fichier situé à l'emplacement spécifié par $file_path.

Exemple d'utilisation :


$file_path = "path/to/your/file.txt";
delete_file($file_path);
