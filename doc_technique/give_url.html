Documentation Technique de la Fonction give_url

Description Générale

La fonction give_url retourne le nom du fichier courant sans son chemin. Elle utilise la variable globale $_SERVER['PHP_SELF'] pour obtenir le chemin du script actuel et retourne le nom du fichier après avoir supprimé le chemin.

Fonction

give_url()

Retourne le nom du fichier actuel sans le chemin.
