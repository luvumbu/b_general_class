Documentation Technique de la Classe Data_send_class

Description Générale

La classe Data_send_class est utilisée pour gérer l'envoi de données à un serveur en utilisant une classe JavaScript nommée Information. Elle permet de stocker des paires de valeurs et d'envoyer ces données à l'aide d'un script JavaScript.

Propriétés

  • $array_add : Tableau pour stocker les paires de valeurs à envoyer.
  • $src_push : Source vers laquelle les données seront envoyées.

Méthodes

__construct($src_push)

Constructeur de la classe. Initialise la source pour l'envoi des données.

Exemple d'utilisation :


$dataSend = new Data_send_class("url_de_destination");

voir($info)

Méthode pour examiner les informations fournies et ajouter celles correspondant à un type "input" au tableau des données à envoyer.

Exemple d'utilisation :


$dataSend->voir([
    ["nomInput", "input"],
    ["submitButton", "button"]
]);

set_array_add($var_nam, $value)

Ajoute une paire de valeurs au tableau des données à envoyer.

Exemple d'utilisation :


$dataSend->set_array_add("nomInput", "input");

get_array_add()

Retourne le tableau des données à envoyer.

Exemple d'utilisation :


$arrayAdd = $dataSend->get_array_add();

push_send_class()

Prépare et envoie les données à l'aide de la classe JavaScript Information.

Exemple d'utilisation :


$dataSend->push_send_class();
