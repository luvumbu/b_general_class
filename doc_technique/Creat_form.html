Documentation Technique de la Classe Creat_form

Description Générale

La classe Creat_form permet de générer dynamiquement des formulaires HTML et leurs éléments enfants via du code PHP et JavaScript. Elle inclut des méthodes pour ajouter des champs de formulaire, gérer les attributs des éléments HTML, et manipuler des fichiers associés, notamment pour stocker des configurations ou des interactions avec des bases de données.

Propriétés

  • $name_form : Nom du formulaire.
  • $name_type : Type d'élément HTML (ex. : 'form', 'div').
  • $child_info : Tableau contenant les informations des éléments enfants.
  • $array_select_Atribut : Tableau des attributs sélectionnés.
  • $nom_dossier : Nom du dossier où le fichier est stocké.
  • $nom_fichier : Nom du fichier à créer.
  • $path : Chemin complet du fichier.
  • $config_bool : Booléen pour la configuration.
  • $path_config : Chemin vers le fichier de configuration.
  • $bool_config : Booléen pour vérifier la configuration.
  • $config_dbname : Nom de la base de données.
  • $config_password : Mot de passe de la base de données.
  • $databaseHandler_verif : Vérification du gestionnaire de base de données.

Méthodes

__construct($name_form, $name_type, $para)

Constructeur de la classe. Initialise le formulaire avec un nom et un type d'élément HTML, et génère le code JavaScript pour créer et ajouter l'élément HTML au DOM.

Exemple d'utilisation :


$monFormulaire = new Creat_form("monFormulaire", "form", "");

add_child($name, $type)

Ajoute un élément enfant au formulaire en créant un nouvel élément HTML avec un ID et l'ajoutant comme enfant de l'élément formulaire principal.

Exemple d'utilisation :


$monFormulaire->add_child("nomInput", "input");
$monFormulaire->add_child("submitButton", "button");

child_setAtribut($id, $name_atribute, $value_atribute)

Définit un attribut spécifique ou le contenu (innerHTML) d'un enfant du formulaire.

Exemple d'utilisation :


$monFormulaire->child_setAtribut("nomInput", "type", "text"); // Définit l'attribut type="text" pour l'input
$monFormulaire->child_setAtribut("nomInput", "placeholder", "Entrez votre nom"); // Définit l'attribut placeholder="Entrez votre nom"
$monFormulaire->child_setAtribut("submitButton", "", "Envoyer"); // Définit le contenu texte du bouton

construct_setAtribut($name_atribute, $value_atribute)

Définit un attribut ou le contenu de l'élément parent (formulaire).

Exemple d'utilisation :


$monFormulaire->construct_setAtribut("method", "POST"); // Définit l'attribut method="POST" pour le formulaire

select_Atribut($value, $nom_dossier, $nom_fichier)

Configure les attributs de sélection et crée un fichier PHP avec le contenu basé sur les attributs spécifiés.

Exemple d'utilisation :


$monFormulaire->select_Atribut("input", "nom_dossier", "nom_fichier.php");

add_child_array($list_array)

Ajoute plusieurs éléments enfants à partir d'un tableau de valeurs.

Exemple d'utilisation :


$monFormulaire->add_child_array([
    ["nomInput", "input"],
    ["submitButton", "button"]
]);

select_Atribut_function()

Crée une fonction JavaScript pour gérer les attributs des éléments sélectionnés.

Exemple d'utilisation :


$monFormulaire->select_Atribut_function();
