Fonction CheckFileExists

La fonction CheckFileExists permet de vérifier si un fichier existe à un emplacement donné. Elle prend en paramètre le chemin du fichier et renvoie un booléen indiquant si le fichier est présent ou non.

Signature de la fonction

function CheckFileExists($filePath) {
    if (file_exists($filePath)) {
        return true; // Le fichier existe
    } else {
        return false; // Le fichier n'existe pas
    }
}

Paramètre

  • $filePath (string) : Chemin du fichier que vous voulez vérifier. Par exemple, '/path/to/file.txt'.

Valeur de retour

  • (bool) : true si le fichier existe, false sinon.

Fonctionnement

La fonction utilise file_exists(), une fonction native de PHP, pour vérifier si le fichier existe à l'emplacement spécifié. Si le fichier est trouvé, la fonction retourne true, sinon elle retourne false.

Exemple d'utilisation

$filePath = '/path/to/file.txt';

if (CheckFileExists($filePath)) {
    echo "Le fichier existe.";
} else {
    echo "Le fichier n'existe pas.";
}

Fonction native utilisée

  • file_exists(string $filePath) : Vérifie si un fichier ou un dossier existe à l'emplacement spécifié.
