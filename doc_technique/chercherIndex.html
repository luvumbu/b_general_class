Fonction chercherIndex

La fonction chercherIndex permet de trouver l'index ou la clé d'une valeur spécifique dans un tableau. Elle utilise la fonction native PHP array_search pour effectuer cette recherche.

Signature de la fonction

function chercherIndex($tableau, $valeur) {
    $index = array_search($valeur, $tableau);

    if ($index !== false) {
        return $index;
    } else {
        return false;
    }
}

Paramètres

  • $tableau (array) : Le tableau dans lequel vous voulez rechercher la valeur.
  • $valeur (mixed) : La valeur que vous souhaitez rechercher dans le tableau.

Valeur de retour

  • (int|string|bool) : Retourne l'index ou la clé de la valeur si elle est trouvée. Si la valeur n'est pas trouvée, la fonction retourne false.

Fonctionnement

La fonction chercherIndex utilise array_search pour rechercher la valeur dans le tableau. Si la valeur est trouvée, elle retourne l'index ou la clé. Si la valeur n'est pas trouvée, elle retourne false.

Exemple d'utilisation

$tableau = array("pomme", "banane", "orange", "fraise");
$valeur_a_chercher = "orange";

$resultat = chercherIndex($tableau, $valeur_a_chercher);

if ($resultat !== false) {
    echo "L'index de '$valeur_a_chercher' est $resultat.";
} else {
    echo "'$valeur_a_chercher' n'a pas été trouvé dans le tableau.";
}

Fonction native utilisée

  • array_search(mixed $value, array $array) : Recherche une valeur dans un tableau et retourne la première clé correspondante si trouvée. Retourne false si la valeur n'existe pas.
