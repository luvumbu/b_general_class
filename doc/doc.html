Documentation de la classe Creat_form

La classe Creat_form permet de créer dynamiquement des formulaires HTML et leurs éléments enfants à l'aide de PHP et de JavaScript. Cette documentation explique chaque méthode de la classe en détail pour les débutants.

Constructeur

__construct($name_form, $name_type, $para)

Le constructeur initialise un nouvel objet Creat_form en créant un élément HTML spécifié.

  • $name_form : Le nom et l'ID de l'élément formulaire.
  • $name_type : Le type d'élément HTML à créer (ex : "form").
  • $para : L'ID de l'élément parent où le formulaire sera inséré. Si vide, le formulaire est ajouté directement au body.

Exemple :

// Créer un formulaire avec ID 'monFormulaire' ajouté au body
$form = new Creat_form("monFormulaire", "form", "");

Méthodes

add_child($name, $type)

Cette méthode ajoute un nouvel élément enfant au formulaire.

  • $name : L'ID de l'élément enfant à créer.
  • $type : Le type de l'élément enfant (ex : "input", "button").

Exemple :

// Ajouter un champ de texte avec l'ID 'nomInput'
$form->add_child("nomInput", "input");

child_setAtribut($id, $name_atribute, $value_atribute)

Cette méthode permet de définir un attribut spécifique ou le contenu d'un élément enfant.

  • $id : L'ID de l'élément enfant cible.
  • $name_atribute : Le nom de l'attribut à définir (ex : "type", "placeholder").
  • $value_atribute : La valeur de l'attribut ou du contenu.

Exemple :

// Définir l'attribut 'type' pour l'élément 'nomInput'
$form->child_setAtribut("nomInput", "type", "text");

// Définir un placeholder pour l'input
$form->child_setAtribut("nomInput", "placeholder", "Entrez votre nom");

construct_setAtribut($name_atribute, $value_atribute)

Cette méthode définit un attribut pour l'élément parent (le formulaire).

  • $name_atribute : Le nom de l'attribut à définir.
  • $value_atribute : La valeur de l'attribut.

Exemple :

// Définir la méthode d'envoi du formulaire à POST
$form->construct_setAtribut("method", "POST");

add_child_array($list_array)

Cette méthode permet d'ajouter plusieurs éléments enfants au formulaire en utilisant un tableau.

  • $list_array : Un tableau contenant les spécifications des éléments enfants.

Exemple :

// Ajouter plusieurs éléments enfants au formulaire
$form->add_child_array([
    ["nomInput", "input"],
    ["submitButton", "button"]
]);

Exemple complet

Voici un exemple complet montrant comment utiliser cette classe pour créer un formulaire simple :

<?php
// Créer un formulaire avec ID 'monFormulaire'
$form = new Creat_form("monFormulaire", "form", "");

// Ajouter un champ de texte (input) au formulaire
$form->add_child("nomInput", "input");

// Ajouter un bouton de soumission au formulaire
$form->add_child("submitButton", "button");

// Définir les attributs pour les enfants
$form->child_setAtribut("nomInput", "type", "text");
$form->child_setAtribut("nomInput", "placeholder", "Entrez votre nom");
$form->child_setAtribut("submitButton", "", "Envoyer");

// Définir la méthode d'envoi du formulaire
$form->construct_setAtribut("method", "POST");
?>

Note : Cette classe génère du code JavaScript pour créer et manipuler dynamiquement les éléments HTML. Assurez-vous que JavaScript est activé dans le navigateur pour que ces fonctions s'exécutent correctement.
