Documentation de la classe Creat_form

La classe Creat_form permet de créer dynamiquement des formulaires HTML et leurs éléments enfants à l'aide de PHP et de JavaScript. Cette documentation explique chaque méthode de la classe en détail, idéale pour les débutants en développement web.

Constructeur

__construct($name_form, $name_type, $para)

Le constructeur initialise un nouvel objet Creat_form en créant un élément HTML spécifié.

  • $name_form : Le nom et l'ID de l'élément formulaire.
  • $name_type : Le type d'élément HTML à créer (par exemple, form, div, input).
  • $para : L'ID de l'élément parent où cet élément sera ajouté. Si vide, l'élément est ajouté directement au corps du document.

Méthodes

add_child($name, $type)

Ajoute un élément enfant au formulaire.

  • $name : L'ID de l'élément enfant.
  • $type : Le type d'élément HTML pour l'enfant (par exemple, input, label).

child_setAtribut($id, $name_atribute, $value_atribute)

Définit un attribut ou le contenu d'un élément enfant spécifique.

  • $id : L'ID de l'élément enfant.
  • $name_atribute : Le nom de l'attribut à définir. Si vide, définit le contenu innerHTML de l'élément.
  • $value_atribute : La valeur de l'attribut ou du contenu.

construct_setAtribut($name_atribute, $value_atribute)

Définit un attribut ou le contenu de l'élément parent (le formulaire principal).

  • $name_atribute : Le nom de l'attribut à définir pour l'élément parent. Si vide, définit le contenu innerHTML de l'élément parent.
  • $value_atribute : La valeur de l'attribut ou du contenu.

add_child_array($list_array)

Ajoute un groupe d'éléments enfants au formulaire à partir d'un tableau d'instructions.

  • $list_array : Un tableau contenant les détails des éléments enfants à ajouter. Chaque sous-tableau doit suivre le format [$id, $type, $attribut].

select_Atribut($value)

Sélectionne les éléments enfants ayant un attribut spécifique.

  • $value : La valeur de l'attribut à rechercher parmi les éléments enfants.

select_Atribut_function($test)

Crée une fonction JavaScript pour surveiller les interactions utilisateur sur les éléments sélectionnés et déclencher des actions spécifiques.

  • $test : Un paramètre pour tester et déclencher des actions basées sur les valeurs des attributs sélectionnés.

Exemple d'utilisation


// Créer un nouveau formulaire
$monFormulaire1 = new Creat_form("form_p_name", "form", "");

// Définir un attribut pour le formulaire
$monFormulaire1->construct_setAtribut("class", "monFormulaire1");

// Ajouter des champs texte et des boutons au formulaire
$monFormulaire1->add_child_array([
    ["input_1", "input", "placeholder='Entrer votre nom'"],
    ["submitButton", "button", "type='submit'"]
]);

// Sélectionner les éléments de type "input"
$monFormulaire1->select_Atribut("input");

// Ajouter une fonction de surveillance pour les éléments sélectionnés
$monFormulaire1->select_Atribut_function("input");
